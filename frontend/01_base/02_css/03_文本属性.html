文本颜色

文本颜色使用 color 关键字设置，有如下几种设置方式：

  • color: value
  • color: #六位十六进制数（rgb）
  • color: rgb(0~255, 0~255, 0~255)
  • color: rgb(0~100%, 0~100%, 0~100%)

文本对齐

使用 text-align 关键字设置，属性值有：

  • center
  • left
  • right

文本装饰

关键字：text-decoration 属性值：none 、 underline 、 overline 、 line-through

可使用 none 属性值去除 <a> 标签中自带的下划线样式 下划线 上划线 删除线

文本缩进

关键字：text-indent 属性值单位：绝对缩进单位 -> px；相对缩进单位 -> em（相对于当前元素的字体大小）小娃娃，你可懂了？

行间距

关键字： line-height 单位：px

行间距由 上间距、文本高度、下间距 三部分组成。测量时，选取相邻两行中上一行的最下沿到下一行的最下沿间的距离

css 样式引入方式

内部（内嵌）方式

将 css 部分集中写入到 <style> 标签中，实现了结构和样式的部分分离

行内样式

格式：在要修改样式的标签中添加属性 style，如 <p style="color: purple"></p>

适用于修改简单样式

外部方式

步骤如下：

  1. 建立相应的 css 文件，写入样式代码，不用再写 <style></style> 标签
  2. 在 html 文件的 <head></head> 标签中写入 <link rel="stylesheet" href="css 文件路径">
