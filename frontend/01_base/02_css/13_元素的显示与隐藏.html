display

语法: display: none|block;
none 会隐藏元素，且不再占有位置；block 除了将元素转化为块级盒子外，还会显示盒子

visibility

语法： visibility: visible|hidden
hidden 会隐藏元素，但任占有原来的位置

overflow

语法： overflow: visible|hidden|scroll|auto；
auto 会按需来决定是否显示滚动条

什么是HTML： 1.HTML 指的是超文本标记语言 (Hyper Text Markup Language) ，它是用来描述网页的一种语言。 2.HTML 不是一种编程语言，而是一种标记语言 (markup language)。 标记语言是一套标记标签 (markup tag)。 网页的总结： 1.网页是图片、链接、文字、声音、视频等元素组成, 其实就是一个html文件(后缀名为html) 2.网页生成制作: 有前端人员书写 HTML 文件, 然后浏览器打开,就能看到了网页. 3.HTML:超文本标记语言, 用来制作网页的一门语言. 有标签组成的. 比如 图片标签 链接标签 视频标签等…
