字体

一般可在 body 标签中设置整体属性，对于特殊标签再调整

尽量使用浏览器中的常见字体

多个单词组成的字体名一般将其使用引号引起来

使用多个字体时，各字体间使用英文逗号隔开

谷歌浏览器的默认字体为微软雅黑（Microsoft YaHei）

文字大小

谷歌浏览器中默认字体大小为 16 像素，不同的浏览器会有不同的默认大小，一般需要明确指定

文字加粗

font-weight 共有 normal 、 bold 、 bolder 、lighter 属性值， 使用 normal 可去除标题自带的加粗样式

其中 nomal 等价于 400， bold 等价于 700

字体样式

font-style 中的 normal 属性可使倾斜标签 <em> 或 <i> 中的内容不倾斜

复合属性

可将 font-style 、 font-weight 、 font-size/line-height 、 font-family 四个属性用 font 关键字替代，给属性值时必须按上述关键字出现的顺序给出

各属性值之间使用空格隔开，其中 font-style 和 font-weight 为可选值，font-size 和 font-family 为必须值
