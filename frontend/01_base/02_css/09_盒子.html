边框

border-width -> 设置边框的宽度；border-style -> 设置边框的样式，属性值有 solid（实线）、dotted（点线）、dashed（虚线）；boeder-color -> 设置边框的颜色

border-collapse -> 合并相邻的边框，常在表格单元格间使用

在给盒子添加边框宽高时，添加的部分会往外扩展，增大盒子

内边距

语法：padding-left|right|top|bottom|: ...px;

复合边距

值的个数 描述
padding: 5px; 上下左右的内边距均为 5px
padding: 5px 10px; 上下内边距为 5px ，左右内边距为 10px
padding: 5px 10px 15px; 上内边距为 5px ， 左右内边距为 10px ， 下内边距为 15px
padding: 5px 10px 15px 20px 上内边距为 5px ， 右内边距为 10px ， 下内边距为 15px ，左内边距为 20px

内边距也会撑大盒子的大小。但未给盒子本身指定宽高时，则 padding 的值不会撑开盒子

外边距

语法同 padding

利用外边距可实现盒子在浏览器中居中对齐，但必须满足两个条件：

  • 盒子设置了宽度
  • 盒子的左右外边距设置为 auto （margin-left: auto; margin-right: auto; | margin: auto; | margin: 0 auto;）

将行内块元素居中对齐，可在其父元素中添加 text-align: center 属性

相邻外边距在垂直方向上会发生合并的现象。解决方案：尽量只给一个盒子添加外边距

当嵌套元素的上外边距都有值时，父元素会发生塌陷。解决方案：

  • 为父元素定义上外边框/内边距
  • 为父元素添加 overflow:hidden

清除内外边距

网页元素很多都有不同的内外边距，而且不同的浏览器默认值不同，在布局前要清楚所有元素的内外边距： * {margin: 0px; padding: 0px;}

行内元素为了照顾兼容性，尽量只设定左右内外边距，不要设定上下内外边距（不起作用）。但转换为块级元素就可以了

圆

在正方形盒子中添加样式 border-radius: 宽度的一半|50%;

圆角矩形

在矩形盒子中添加样式 border-radius: 高度的一半;

任意圆角

格式： border-radius: apx bpx cpx dpx; 分别作用于从左上角顺时针旋转的圆角。属性值给两个时，作用于两对角线角。也可以单独设置：border-top-left-radius: apx;

盒子阴影

值 描述
h-shadow 必须值。水平阴影的位置，允许负值。
v-shadow 必须值。垂直阴影的位置，允许负值。
blur 可选值。模糊距离。
spread 可选值。阴影的尺寸。
color 可选值。阴影的颜色。
inset 可选值。将外部阴影（outset）改为内部阴影。

注意：默认为外阴影（outset）效果，但不能写上这个单词。否则会失效；阴影不占用空间，不影响其它盒子的排列。

blur 可选值。模糊距离。
