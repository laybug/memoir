类选择器、属性选择器、伪类选择器的权重均为 0

属性选择器

  • E[attr]：选择 E 标签中带有 attr 属性的元素
  • E[attr="val"]：选择 E 标签中带有 attr 属性且值为 val 的元素
  • E[attr^="val"]: 选择 E 标签中 attr 属性值开头含有 val 的元素
  • E[attr$="val"]: 选择 E 标签中 attr 属性值结尾含有 val 的元素
  • E[attr*="val"]: 选择 E 标签中 attr 属性值中含有 val 的元素

结构伪类选择器

  • 父元素 子元素:first-child ：选择父元素里面第一个叫子元素的标签
  • 父元素 子元素:last-child ：选择父元素里面最后一个叫子元素的标签
  • 父元素 子元素:nth-child(n) ：选择父元素里面第 n 个叫子元素的标签
  1. nth-child(n) 其中的 n 可以是数字、关键字、公式
  2. 数字：只能为字母 n ，n 从 0 开始每次自增 1 ，当遇到 0 和第 n+1 个子元素时自动忽略，表示选中父元素中的所有子元素
  3. 关键字：even ： 偶数子元素； odd 奇数子元素
  4. 公示：2n -> 偶数； 2n+1 -> 奇数； 5n -> 5 的倍数； n+5->从第 5 个开始以后（含第 5 个）的子元素；-n+5->从第 5 个开始以前（含第 5 个）的子元素；

E:nth-child(n) 和 E:nth-of-type区别

E:nth-child 先执行 nth-child(n) （给所有子元素排序）再匹配第 n 个子元素是否符合 E 元素

E:nth-of-type(n) 先执行 E （给指定的子元素 E 排序）再匹配 E 元素中是否有符合 n 的子元素

伪元素

代码：E::after|before 。before 在父元素内容的前面创建一个元素；after 在父元素内容的后面创建一个元素

其由 css 代码生成，在文档树中找不到该标签，故称之为伪元素

before 和 after 必须含有 contetn 属性。伪元素选择器的权重和标签选择器一样为 0

盒子大小计算方式

css 默认盒子大小计算方式为 box-sizing: content-box; 加入外边框、内外边距等都会撑大盒子。新增样式 box-sizing: border-box 则不会随外边框等的加大而加大（边距、边框之和小于盒子本身）
